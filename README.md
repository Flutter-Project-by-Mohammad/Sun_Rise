# AtmosScan

</div>

### Your Real-Time Global Weather Companion

AtmosScan is a modern, responsive, and lightweight weather application built with Flutter. It allows users to search for any city worldwide and view detailed, real-time weather information. The app integrates with live REST APIs for accurate data and follows a clean, scalable MVC structure.  
Designed exclusively in **light mode**, AtmosScan delivers a smooth and visually appealing user experience.
